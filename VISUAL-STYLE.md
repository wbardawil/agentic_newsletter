# Visual Style Guide

This document outlines the visual style guidelines for The Transformation Letter.

## Palette

- Primary Accent: #3498db (Blue)
- Secondary Accent: #2ecc71 (Green)
- Text: #2c3e50 (Dark Blue/Gray)
- Background: #ecf0f1 (Light Gray)

## Typography

- Headings: Georgia, serif
- Body: Lato, sans-serif

## Layout Tokens

- Spacing: Consistent use of 8px grid system.
- Borders: Subtle, 1px solid #ddd.

## Imagery

- Style: Clean, modern, professional. Abstract concepts preferred over literal.
- Iconography: Flat, minimalist design.
