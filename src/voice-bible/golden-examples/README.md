# Golden Examples

Place example newsletters here to guide the Writer agent's output quality.

## File Naming Convention

```
YYYY-WW-<lang>.md
```

Examples:
- `2026-07-en.md` — English edition for week 07 of 2026
- `2026-07-es.md` — Spanish edition for week 07 of 2026

## Notes

- This README.md file is ignored by the voice-bible loader.
- Each file should contain a complete, publication-ready newsletter edition.
- Examples are injected into the Writer agent's system prompt as reference material.
